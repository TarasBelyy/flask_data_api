# API сервис для анализа данных


## Описание
REST API для статистической обработки данных, а также для очистки первичных данных.
Доступные методы запроса, эндпоинты, коды корректного выполнения запросов:
```
POST 
/upload      
201
```
```
GET 
/data/stats   
200
```
```
GET 
/data/clean   
200
```

## Инструкция 
Для запуска приложения из консоли наберите:
```
flask --app api run
```
Для обращения к приложению из консоли наберите: 
```
curl -X POST -F "file=@/<file-path>" http://127.0.0.1:5000/upload
```
```
curl http://127.0.0.1:5000/data/stats
```
```
curl http://127.0.0.1:5000/data/clean
```

## Функциональность
- загружаются файлы в формате csv с произвольным количеством столбцов
- производятся вычисления для нахождения средних значений, медиан, матрицы корреляций (Пирсон)
- осуществляется очистка первичных данных от дубликатов
- в случае ошибок передаются соответствующие сообщения и код ошибки: 400, 404, 415, 500

## Требования
Основные внешние библиотеки и фреймворки для Python, используемые в проекте: 
- Flask версии 3.1.3
- pandas версии 3.0.3
- SQLAlchemy версии 2.0.51
- psycopg2-binary версии 2.9.12

Для хранения данных используется PostgreSQL.
